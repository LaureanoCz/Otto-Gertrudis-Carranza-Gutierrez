Proyecto de Robot Multifuncional: "Otto Gertrudis Carranza Gutiérrez"
Descripción de la Propuesta del Proyecto
"Otto Gertrudis Carranza Gutiérrez" es un robot multifuncional diseñado para realizar movimientos como saltar, bailar, caminar y girar. Este innovador robot tiene como objetivo proporcionar entretenimiento y fomentar el aprendizaje en robótica, ofreciendo una experiencia interactiva y educativa para usuarios de todas las edades.

Componentes Utilizados
Electrónicos
Arduino Uno: Microcontrolador que actúa como el cerebro del robot, controlando todos los movimientos y funciones.
Shield para Arduino Uno: Facilita las conexiones y expansión de funciones.
Sensor Ultrasonico HC-SR04: Permite la detección de obstáculos para una navegación segura.
4 Servomotores 180º: Proporcionan la movilidad necesaria para realizar saltos, giros y bailes.
Buzzer de 5V: Emite sonidos que acompañan las rutinas de baile y otras interacciones.
Cables Dupont HH (6 unidades): Conectores que permiten la unión de los componentes eléctricos.
Portapilas para 4 pilas AA: Provee la energía necesaria para el funcionamiento del robot.
Switch o Interruptor de Encendido: Permite encender y apagar el robot fácilmente.
